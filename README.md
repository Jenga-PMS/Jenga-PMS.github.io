# Jenga Documentation

### Live Documentation: https://jenga-pms.github.io/

---

This repository contains the documentation for **[Jenga](https://github.com/Jenga-PMS/Jenga)**,  a project management and issue/ticket tracking tool for development.

The documentation is built with [MkDocs](https://www.mkdocs.org/) using the [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) theme  and is hosted on [GitHub Pages](https://pages.github.com/).
